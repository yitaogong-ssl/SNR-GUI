What This Tool Does

Multi-ROI Input

Upload one CSV per ROI and aggregate SNR by marker.

Robust SNR

Top-signal over bottom-signal strategy with caps, epsilon, and denominator stability checks.

Clear Decision Labels

Per ROI and summary labels: Pass for SNR >= 3, Good for SNR >= 10.

Shareable Output

Export CSVs plus barplot in one ZIP for downstream reporting and archiving.

SNR Interpretation

Pass: SNR >= 3
Good: SNR >= 10

The barplot includes guide lines for both thresholds, and tables include matching boolean columns.

Quick Start

  1. Clone this repository.
  2. Install dependencies.
  3. Run Streamlit and upload ROI CSV files.
pip install streamlit pandas numpy matplotlib
streamlit run "# snr_gui.py"

Optional cleanup: rename # snr_gui.py to snr_gui.py for simpler CLI usage.

GitHub Pages Deployment

This repo includes a workflow at .github/workflows/deploy-pages.yml that publishes the docs/ folder automatically after push to main.

  1. Push this repository to GitHub.
  2. In repo settings, enable Pages source as GitHub Actions.
  3. Visit your site URL: https://<user>.github.io/<repo>/.
